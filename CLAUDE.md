# Handover Player - 動画レビューツール

Dropbox動画直接再生、タイムコード付きコメント、Premiere/Resolve エクスポート。

## Stack
React 19 + Vite 7 + Tailwind CSS v4 + Supabase (delaxsupabase)
Vercel auto-deploy from main

## Design System
- Tokens: `src/index.css` (CSS custom properties)
- Components: `src/components/ui/` (Button, Card, Input, Badge, etc.)
- Style: Light professional — white #ffffff base, indigo #6366f1 accent, Inter + JetBrains Mono

## Commands
| コマンド | 用途 |
|---------|------|
| `npm run dev` | 開発サーバー |
| `npm run build` | ビルド |
| `npm run lint` | ESLint |
